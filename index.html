About Me

As passionate third-year undergraduate student at Acropolis Institute of Technology and Research, I am on a journey to become a proficient full-stack developer. My academic major in computer science has been a gateway to understanding the intricacies of technology, and it has fueled my curiosity and desire to dive deeper into the world of web development and software engineering.

From an early age, I have been fascinated by how things work, which naturally led me to the field of technology. My initial exposure to programming came through languages like HTML and CSS, which allowed me to create basic web pages. This sparked an interest that quickly evolved as I delved into JavaScript and React, enabling me to build more dynamic and interactive web applications. My proficiency in these technologies has been further enhanced by using Redux for state management and Tailwind CSS for designing responsive and aesthetically pleasing user interfaces. My journey as a developer is marked by hands-on experience with various projects. Notably, I have built an online food delivery application using React, which involved managing complex state and ensuring a seamless user experience. Another significant project was a movie streaming platform integrated with AI recommendations, reflecting my interest in leveraging AI to enhance user engagement. These projects have not only honed my technical skills but also taught me the importance of user-centric design and robust backend infrastructure, even though I primarily focus on front-end development.
